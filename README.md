# PromoFarma
A Spring Boot application designed to manage a shopping cart system, allowing users to add, update, and remove products from their carts, calculate totals, and confirm orders.

Features
路 Products: Add, view, and delete products linked to a seller.
路 Cart Management: Add products to the cart, update quantities, remove products, and clear the cart.
路 Order Confirmation: Confirm cart items for processing.
路 Total Calculation: Calculate the total price of all items in the cart.
